# SINTEF Weather Display

Real-time weather display for digital signage in Trondheim.

## Features

- Real-time weather data from MET.no API
- Animated weather effects (rain, snow, clouds)
- Terminal-style typewriter animation
- Wind chill calculation
- SINTEF branding and color scheme

## Files

- `trondheim-weather.html` - Main weather display
- `weather-display.css` - Styles and animations
- `sintef-blog-display.html` - Blog post display
- `sintef-blog-display.css` - Blog display styles

## Technology

- HTML, CSS, JavaScript
- MET.no LocationForecast API
- Google Fonts (JetBrains Mono)
